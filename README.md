##Library Management System - Data Analysis Project-02
​##Project Overview

**Project Title**: Library Management System  
**Database**: `library_db`

​This project involves managing and analyzing library data, including Books, Branches, Employees, Members, and tracking Issued and Returned statuses. The goal is to demonstrate SQL proficiency through complex queries, data cleaning, and insightful reporting.
​**Database Structure**
​The project is built around the following relational tables:
​Books: Information about titles, authors, and availability.
​Branches: Details about different library locations.
​Employees: Staff information linked to specific branches.
​Members: Library member details and registration info.
​Issued Status: Records of books borrowed by members.
​Return Status: Records of books returned to the library.
